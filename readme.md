# LD Fix User Access

## Description

This plugin allows you to fix user access for LearnDash courses.

## Installation

1. Upload the plugin files to the `/wp-content/plugins/fix-user-access-for-learndash` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

## Usage

### Fix User Access

Go to the 'Users' menu in the WordPress admin panel and click on 'Fix User Access'.

### Fix User Access for Course

Go to the 'Courses' menu in the WordPress admin panel and select the course you want to fix user access for.

Inside the course edit screen, click on the 'Fix User Access' tab.

In the User with Fixed Access field, select the users you want to fix access for.

This is a multi-select field, so you can select multiple users. You can search users by name, username or ID.

After selecting the users, click on the 'Save Changes' button to save the changes.

### Copy Fixed Access Settings

In the Copy Fixed Access Settings field, select the course you want to copy fixed access settings from.

NOTE: You'll only be able to use this feature if at least one course has fixed access users already configured.

## Global Fixed Access

Go to the 'Users' menu in the WordPress admin panel and click on 'Fix LD Access'.

In the Global Fixed Access field, select the users you want to fix access for.

This is a multi-select field, so you can select multiple users. You can search users by name, username or ID.
